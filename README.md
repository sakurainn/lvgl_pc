# lvgl_pc

## Introduction

This is a simple project to use lvgl on PC easily. It will download the dependencies and build lvgl on your PC. Not use libs in the system. So I can build lvgl on my Windows laptop and my Macbook all well.

PS. This project will download source from GitHub. So if you are in China, you need to configure proxy.

## How to use

```bash
git clone https://github.com/sakurainn/lvgl_pc.git
cd lvgl_pc
mkdir build
cd build
cmake ..
make
```
